Growth with Time

Tim Mei, Period 9

Project Description

The project is essentially just a page that provides visuals on a dataset. The dataset is completely modifiable and will work as long as the provided data is a set of numbers.

How it Works

A list of rates inspired by Bankrates and Nerdstreet Wallet are put into a CSV file. It is then read and stored in separate lists within a list. The program then takes two random rates and separates them into three parts: the initial balance, the rate, and the compounds per year. It then takes the compound interest formula and takes the values to generate a new list, one with the values in the table. The values are then graphed and also plugged into another program that will take these values and places them accordingly in the table.
